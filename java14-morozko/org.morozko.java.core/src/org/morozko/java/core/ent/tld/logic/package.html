Tag library personalizzata per gestire comparazione e altre operazioni all' interno di pagine JSP (oilogic).


Tag Descrizione
compare Test di uguaglianza tra due bean.
compareNot Test di non uguaglianza tra due bean.
greaterThan Verifica se di due bean che rappresentano un valore numerico, il primo è maggiore del secondo.
greaterEqual Verifica se di due bean che rappresentano un valore numerico, il primo è maggiore o uguale al secondo.
lesserThan Verifica se di due bean che rappresentano un valore numerico, il primo è minore del secondo.
lesserEqual Verifica se di due bean che rappresentano un valore numerico, il primo è minore o uguale al secondo.



Tag : ompare

Tag che esegue il codice in esso contenuto solo se il risultato della comparazione tra due oggetti restituisce true.

Ad esempio il seguente estratto di codice :
<oilogic:compare name='bean' nameTo='beanTo'>
TEST </oilogic:compare name='bean' nameTo='beanTo'>


E' equivalente a scrivere in una pagina JSP :
<%
   Object obj1 = this.pageContext.findAttribute( "bean" );
   Object obj2 = this.pageContext.findAttribute( "beanTo" );
   if ( obj1.equals( obj2 ) ) {;
    out.print( "TEST" );
   }
%>
compare
attributo descrizione
name Il bean che viene utilizzato come primo membro della comparazione.
[REQUIRED]
property La proprietà del bean identificato dall' attributo name che viene utilizzata per ottenere il primo membro della comparazione. Se questo attributo non viene specificato il bean stesso viene utilizzato.
nameTo Il bean che viene utilizzato come secondo membro della comparazione.
[REQUIRED]
propertyTo La proprietà del bean identificato dall' attributo nameTo che viene utilizzata per ottenere il secondo membro della comparazione. Se questo attributo non viene specificato il bean stesso viene utilizzato.
compareString Questo attributo indica se prima di essere comparati, i due bean vadano trasformati in stringhe ( String.valueOf() ). [true|false]
