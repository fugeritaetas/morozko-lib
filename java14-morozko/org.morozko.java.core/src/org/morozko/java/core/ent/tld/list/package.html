Package for handling java.util.List instances through jsp tags.

Collezioni di classi per la manipolazione istanze di java.util.List tramite tag jsp.

API Reference

Specifiche API
  • define
define
Tag for defining a new instance of java.util.List and putting it in some application map. Tag per la definizione di una nuova istanza di java.util.List e per metterla in una delle mappe dell' applicazione.
Attribute Name Nome Attributo Description Descrizione
id Specifies the name of the attribute that will be made available. Specifica il nome del attributo che sarà reso disponibile.
scope The scope where to search the specified attribute name. If nost specified, all scope are searched in this order : page, request, session, context. Lo scope dove cercare l'attributo con nome specificato. Se non indicato, vengono scandagliati tutti gli scope in questo ordine : page, request, session, context.
name The attribute name to search for. Il nome dell' attributo da cercare.
property The property of the attribute of given name to call. La proprietà dell' attributo con nome dato da richiamare.
toScope The scope where to put the new definition. Lo scope dove inserire la nuova definizione.
